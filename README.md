# Predictive Analytics Using Historical Data

## Project Overview

This project focuses on forecasting trends using historical sales data. Data preprocessing, regression modeling, and prediction analysis were performed to understand future sales patterns.

## Features

* Data Cleaning and Preprocessing
* Missing Value Analysis
* Linear Regression Model
* Sales Prediction
* Model Accuracy Evaluation
* Data Visualization

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Project Files

* predictive_analytics.py
* train.csv
* Screenshots and charts in the images folder

## Outcome

Built a predictive model using historical data and evaluated its performance through R² Score and visualization charts.
